CSS Styling Explained

Learn CSS concepts with interactive examples

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages — for example, to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

CSS Syntax

CSS consists of selectors and declaration blocks:

selector {
  property: value;
}

Common CSS Properties

Color Properties

Control text and background colors:

.element {
  color: red; /* Text color */
  background-color: #ffffff; /* Background color */
}

Box Model

Every element is a box with content, padding, border, and margin:

.box {
  width: 200px;
  height: 100px;
  padding: 20px; /* Space inside the border */
  border: 2px solid black; /* Border around padding */
  margin: 10px; /* Space outside the border */
}

CSS Selectors

Selectors are patterns used to select the element(s) you want to style:

/* Element selector */
p { color: blue; }

/* Class selector */
.className { font-size: 16px; }

/* ID selector */
#idName { margin: 10px; }

/* Attribute selector */
a[target="_blank"] { color: purple; }
